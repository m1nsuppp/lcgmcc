### **3D 벡터 시각화기 (Vector Visualizer)**

**핵심 개념:** 벡터의 덧셈, 내적, 외적

**무엇을 만드나:**

- 3D 공간에 화살표(벡터)들을 그려서 마우스로 조작
- 두 벡터의 합을 실시간 표시
- 내적 결과를 각도나 길이로 시각화
- 외적 결과를 새 벡터(법선)로 시각화

**학습 포인트:**

- `dot`, `cross`의 기하적 의미
- 단위벡터(normalized vector) 감각
- “법선”이라는 개념의 시각적 이해

🧩 **기술 팁:** Three.js로 웹에서 빠르게 구현 가능

→ `ArrowHelper`, `Vector3.add`, `Vector3.cross` 직접 써보기
